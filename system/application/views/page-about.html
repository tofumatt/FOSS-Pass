This site was built in response to the overuse of the Open Source Initiative's license pages by many open-source projects and hosts (including Google Code). While I think the OSI are cool and have no personal/professional beefs with them, their license pages contain a lot of erroneous extra info (comments, layout, links, etc.) if all you want to do is read a license.

This site was created as a simple way to have a permanent URL for your license text (without having to be responsible for that URL forever).

This site also gives you the ability to embed your own info into the license text so the license people read actually reflects your project, instead of just being a generic license with placeholder text.

A license being available on this site does not mean it's OSI-approved â€” check out the OSI's list of licenses by category for all licenses that are OSI-approved. Licenses are on this site if they encompass the spirit of open-source (i.e. you can distribute and edit the source code).

Site Technology

This site was built in PHP, using the CodeIgniter PHP Framework. The source is managed with git and is hosted on GitHub.
